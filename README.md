# SL-COGS-calculator

Manufacturing cost of goods sold (COGS) calculator with cascade yield modeling, sensitivity analysis and what-if scenario comparison.

## Features

- COGS breakdown by cost category (material, direct labor, variable overhead, fixed overhead, scrap).
- Cascade yield simulation across multi-step manufacturing processes.
- Sensitivity analysis showing cost impact of individual parameter variations.
- What-if scenario comparison with side-by-side Plotly charts.
- JSON import/export for product cost structures.
- Bilingual interface (FR/EN).

## Installation

```bash
git clone https://github.com/Erikeo29/SL-COGS-calculator.git
cd SL-COGS-calculator
pip install -r requirements.txt
```

## Usage

```bash
streamlit run app.py
```
