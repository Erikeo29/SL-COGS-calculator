# À propos

## Calculateur COGS manufacturing

Outil interactif de simulation des coûts de fabrication (COGS) pour produits
manufacturés en plusieurs étapes séquentielles.

---

## Stack technique

| Composant | Technologie |
|-----------|-------------|
| Frontend | [Streamlit](https://streamlit.io) |
| Graphiques | [Plotly](https://plotly.com) |
| Calcul | [NumPy](https://numpy.org), [Pandas](https://pandas.pydata.org) |
| Authentification | [Streamlit-Authenticator](https://github.com/mkhorasani/Streamlit-Authenticator) |
| Déploiement | Streamlit Community Cloud |

---

## Modèle de calcul

Le simulateur implémente un modèle de **cascade multi-étapes** standard en ingénierie
industrielle, où chaque étape de fabrication ajoute des coûts (matériaux, main d'œuvre,
frais généraux) et où les pertes de rendement augmentent le coût unitaire des pièces
conformes.
