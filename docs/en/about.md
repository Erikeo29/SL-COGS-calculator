# About

## COGS manufacturing calculator

Interactive simulation tool for manufacturing costs (COGS) for products
manufactured in multiple sequential steps.

---

## Technical stack

| Component | Technology |
|-----------|------------|
| Frontend | [Streamlit](https://streamlit.io) |
| Charts | [Plotly](https://plotly.com) |
| Computation | [NumPy](https://numpy.org), [Pandas](https://pandas.pydata.org) |
| Authentication | [Streamlit-Authenticator](https://github.com/mkhorasani/Streamlit-Authenticator) |
| Deployment | Streamlit Community Cloud |

---

## Calculation model

The simulator implements a standard **multi-step cascade model** used in industrial
engineering, where each manufacturing step adds costs (materials, labor, overhead)
and yield losses increase the unit cost of conforming parts.
